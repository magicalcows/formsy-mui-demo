# formsy-mui-demo
A simple Meteor app to demo formsy-material-ui


## Usage

[Install meteor](https://www.meteor.com/install)
Clone this repo: `git clone https://github.com/mbrookes/formsy-mui-demo.git`
`cd formsy-mui-demo`
`meteor`
`open http://localhost:3000`
